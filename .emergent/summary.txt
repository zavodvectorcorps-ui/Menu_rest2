<analysis>**original_problem_statement:**
Пользователь хочет создать веб-приложение Личный кабинет ресторана, похожее на сервис LunchPad.

**Ключевые требования:**
1.  **Админ-панель для владельца/менеджера ресторана:**
    *   **Разделы:** Профиль, Меню, Заказы, Настройки, Справочный центр, Помощь.
    *   **Профиль:** Отображение информации о ресторане и виджетов со статистикой (просмотры, вызовы персонала, заказы, сотрудники). Промо-блок для онлайн-редактора меню.
    *   **Меню:** Управление категориями и позициями (название, описание, цена, статус, спец. отметки). Возможность привязать дизайн.
    *   **Заказы:** Таблица с текущими и прошлыми заказами.
    *   **Настройки:** Управление функциями, информацией о ресторане и интеграциями (Telegram).
2.  **Клиентское меню для гостей:**
    *   Доступ по QR-коду/ссылке для каждого стола.
    *   Адаптивность под мобильные устройства.
    *   Отображение категорий и блюд из админ-панели.
    *   Опциональные функции: заказ со стола, вызов персонала.
3.  **Связь между админ-панелью и клиентским меню:**
    *   Все данные в клиентском меню (цены, наличие) должны быть актуальными.
    *   Действия гостя (заказы, вызовы) должны отражаться в статистике и соответствующих разделах админ-панели.
4.  **Пользовательские уточнения:**
    *   Авторизация не нужна. Клиент сканирует QR-код и видит меню.
    *   Нужны уникальные ссылки/коды для столов для корректной работы вызова официанта.
    *   Основной язык: русский.
    *   Цветовая схема: мятные и коричневые цвета, с возможностью переключения темы в админке.
    *   Интеграция с Telegram-ботом.
    *   Использовать предоставленные пользователем данные меню.
    *   Убрать брендинг Made with Emergent.
    *   Реализовать drag-and-drop для сортировки категорий и позиций меню.
    *   **Последний крупный запрос:**
        1.  Разделить клиентское меню на 3 редактируемых блока (например, Гастрономическое, Барное, Кальянное). Категории должны привязываться к этим блокам в админке.
        2.  Добавить возможность создавать рекламные баннеры (картинка на всю ширину) как позиции меню, но без цены.
        3.  Валюта: белорусский рубль (BYN).
        4.  Кнопка Вызов должна открывать модальное окно с кастомными типами вызовов (например, Вызвать официанта, Попросить счет), которые настраиваются в админке и привязаны к разным уведомлениям в Telegram.
