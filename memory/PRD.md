# PRD: Личный кабинет ресторана

## Дата создания: 2026-01-26
## Последнее обновление: 2026-03-10

## Оригинальное ТЗ
Веб-приложение "Личный кабинет ресторана" по типу LunchPad. Админка + клиентское меню по QR-коду.

## Реализовано

### Мультитенантность и авторизация
- JWT, роли суперадмин/менеджер, переключатель ресторанов, управление пользователями

### Telegram-бот
- Раздел в админке, webhook, уведомления при вызовах/заказах, подписчики

### Импорт меню
- .data (LunchPad) и .json форматы
- **Выбор режима: полное обновление (удаляет старое) или дополнение (добавляет к существующему)**

### Ярлыки для блюд (NEW)
- CRUD ярлыков: название + цвет, палитра предустановленных цветов, предпросмотр
- Назначение ярлыков на позиции в диалоге редактирования
- **Отображение ярлыков в клиентском меню** (card + compact режимы)
- Удаление ярлыка автоматически убирает его со всех позиций

### Управление меню
- CRUD категорий/позиций, drag-and-drop, загрузка изображений, два режима отображения

### Логотип
- UtensilsCrossed (вилка/нож) вместо листочка на странице авторизации

### Аналитика, настройки, столы, QR-коды, тёмная тема
### Футер "Made by Knyazev" + MK

## Технический стек
- **Frontend**: React, Tailwind CSS, Shadcn/UI, dnd-kit
- **Backend**: FastAPI, Motor, JWT, qrcode, httpx
- **Database**: MongoDB
- **Integrations**: Telegram Bot API

## Ключевые API
- `/api/restaurants/{id}/labels` — CRUD ярлыков
- `/api/restaurants/{id}/import-file?mode=replace|append` — импорт файлов
- `/api/restaurants/{id}/import-menu` — импорт JSON с mode в body
- `/api/public/menu/{code}` — публичное меню с labels
