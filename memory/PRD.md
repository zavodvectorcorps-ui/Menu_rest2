# PRD: Личный кабинет ресторана

## Дата создания: 2026-01-26
## Последнее обновление: 2026-03-10

## Оригинальное ТЗ
Веб-приложение "Личный кабинет ресторана" по типу LunchPad для партнёров. Админка для владельца/менеджера + клиентское меню по QR-коду.

## Реализовано

### Мультитенантность и авторизация
- JWT-авторизация, роли суперадмин/менеджер
- Переключатель ресторанов, управление пользователями

### Telegram-бот
- Раздел в админке для каждого ресторана (ввод токена, подписчики)
- Webhook, уведомления при вызовах персонала и заказах

### Импорт меню
- Импорт из **.data** файлов (формат LunchPad) — парсинг HTML, цен, изображений, режима отображения
- Импорт из **.json** файлов
- Кнопка "Импорт меню" принимает оба формата

### Управление меню
- CRUD категорий/позиций, drag-and-drop, загрузка изображений
- Два режима отображения: карточка / компактный список

### Аналитика, настройки, столы, QR-коды
### Клиентское меню по QR-коду с тёмной темой
### Футер "Made by Knyazev" + MK на всех страницах
### Удалена дублирующая вкладка "Интеграции" из настроек

## Технический стек
- **Frontend**: React, Tailwind CSS, Shadcn/UI, dnd-kit
- **Backend**: FastAPI, Motor, JWT, qrcode, httpx
- **Database**: MongoDB
- **Integrations**: Telegram Bot API

## API Endpoints
- `/api/restaurants/{id}/import-file` — POST загрузка .data/.json файлов (NEW)
- `/api/restaurants/{id}/import-menu` — POST импорт из JSON body
- `/api/restaurants/{id}/telegram-bot` — GET/PUT/DELETE
- `/api/telegram/webhook/{restaurant_id}` — POST (public)
