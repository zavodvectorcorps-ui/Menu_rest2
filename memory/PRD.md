# PRD: Личный кабинет ресторана

## Дата создания: 2026-01-26
## Последнее обновление: 2026-03-10

## Оригинальное ТЗ
Веб-приложение "Личный кабинет ресторана" по типу LunchPad для партнёров. Админка для владельца/менеджера + клиентское меню по QR-коду.

## Реализовано

### Мультитенантность и авторизация
- JWT-авторизация с логином/паролем
- Суперадмин (admin/220066) с полным доступом
- Роли: Суперадмин (все рестораны), Менеджер (выбранные рестораны)
- Переключатель ресторанов в сайдбаре админки
- Страница управления пользователями (только для суперадмина)

### Telegram-бот (NEW)
- Раздел "Telegram-бот" в админке для каждого ресторана
- Ввод токена от @BotFather, валидация токена через Telegram API
- Автоматическая установка webhook при подключении бота
- Подписка персонала через /start в боте
- Управление подписчиками (просмотр, удаление)
- Уведомления при:
  - Вызове персонала (официант, кальянный мастер)
  - Запросе счёта
  - Новом заказе (с деталями: стол, позиции, сумма)
- Каждый ресторан имеет свой отдельный бот

### Аналитика
- Страница аналитики с графиками
- Метрики: просмотры меню, заказы, выручка, вызовы персонала
- Периоды: 7/30/90 дней

### Управление меню
- CRUD категорий и позиций с drag-and-drop
- Загрузка изображений для позиций меню
- Два режима отображения: карточка / компактный список
- Импорт меню из JSON
- Длинные названия категорий корректно переносятся

### Настройки и столы
- Генерация QR-кодов для столов
- Улучшенная кнопка QR с текстовой меткой

### Клиентское меню
- Доступ по /menu/:tableCode (без авторизации)
- Тёмная тема на мобильных, разделы "Еда", "Напитки", "Кальяны"
- Онлайн-заказ и вызов персонала

### Футер (NEW)
- "Made by Knyazev" с логотипом MK на всех страницах (админка + клиентское меню)

## Технический стек
- **Frontend**: React, Tailwind CSS, Shadcn/UI, dnd-kit
- **Backend**: FastAPI, Motor, JWT, qrcode, Pillow, httpx
- **Database**: MongoDB
- **Integrations**: Telegram Bot API (direct HTTP via httpx)

## API Endpoints (ключевые)
- `/api/auth/login` — авторизация
- `/api/restaurants/{id}/telegram-bot` — GET/PUT/DELETE управление ботом
- `/api/telegram/webhook/{restaurant_id}` — POST webhook (public)
- `/api/restaurants/{id}/import-menu` — импорт JSON
- `/api/public/staff-calls` — вызов персонала + Telegram уведомление
- `/api/public/orders` — заказ + Telegram уведомление
