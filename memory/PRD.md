# PRD: Личный кабинет ресторана

## Дата создания: 2026-01-26

## Оригинальное ТЗ
Веб-приложение "Личный кабинет ресторана" по типу LunchPad для партнёров. Админка для владельца/менеджера + клиентское меню по QR-коду.

## Пользовательские персоны
1. **Владелец/менеджер ресторана** - управляет меню, заказами, настройками
2. **Гость ресторана** - просматривает меню, делает заказы, вызывает персонал

## Основные требования
- Без авторизации для гостей - уникальные коды для столов
- Только русский язык
- Мятно-коричневая цветовая схема (#5DA9A4, #8D6E63)
- Переключение темы из админки
- Тестовые данные "Мята Спортивная"

## Реализовано ✅

### Бэкенд (FastAPI + MongoDB)
- REST API с 20+ эндпоинтами
- Модели: Restaurant, Category, MenuItem, Table, Order, StaffCall, Employee, Settings
- Статистика: просмотры меню, вызовы персонала, заказы
- Автоматическое заполнение тестовыми данными (seed)

### Админка (React)
- **Профиль**: виджеты статистики, информация о ресторане, промо-блок редактора
- **Меню**: CRUD категорий и позиций, поиск, бейджи (хит, новинка, острое, акция)
- **Заказы**: таблица заказов, вкладка вызовов персонала, обновление статусов
- **Настройки**: 
  - Функции (переключатели модулей)
  - О ресторане (название, адрес, контакты, режим работы)
  - Столы (CRUD, уникальные коды, QR-ссылки)
  - Сотрудники (CRUD с Telegram ID)
  - Интеграции (Telegram bot)
  - Оформление (тема, цвета)
- **Справочный центр**: FAQ с поиском и категориями
- **Поддержка**: форма обращения

### Клиентское меню
- Доступ по /menu/:tableCode
- Мобильная адаптивность
- Категории блюд с горизонтальным скроллом
- Карточки блюд с бейджами
- Добавление в корзину
- Оформление заказа с комментарием
- Вызов официанта
- Контактная информация ресторана

## Технический стек
- **Frontend**: React, Tailwind CSS, Shadcn/UI, Lucide Icons
- **Backend**: FastAPI, Motor (async MongoDB)
- **Database**: MongoDB
- **Fonts**: Manrope (headings), Inter (body)
